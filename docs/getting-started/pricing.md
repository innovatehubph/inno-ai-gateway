## Cost Examples

### Example 1: Basic Chatbot

**Usage:**
- 1,000 chat requests/day
- Average 500 tokens per request

**Cost:**
```
1,000 × 30 days = 30,000 requests/month
30,000 × 500 tokens = 15,000,000 tokens
15,000,000 × ₱0.15/1K = ₱2,250/month

Recommended: Startup Plan (₱1,499 + ₱750 overage) = ₱2,249/month
```

### Example 2: Image Generation App

**Usage:**
- 500 images/month using FLUX Dev
- Social media marketing

**Cost:**
```
500 images × ₱0.18 = ₱90/month

Recommended: Developer Plan (₱499) with plenty of room to grow
```

### Example 3: Enterprise RAG System

**Usage:**
- 10M tokens/month (embeddings)
- 100K chat requests
- Custom model fine-tuning

**Cost:**
```
10,000,000 × ₱0.015/1K = ₱150 (embeddings)
100,000 × 1,000 tokens × ₱0.15/1K = ₱15,000 (chat)
Total: ₱15,150/month

Recommended: Enterprise Plan (₱4,999) + credits
```
