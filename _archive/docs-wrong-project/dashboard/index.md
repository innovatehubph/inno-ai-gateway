# Dashboard

The InnoAI Dashboard provides a comprehensive interface for managing campaigns, users, webhooks, and bot flows.

## Dashboard Sections

### Home
Overview of your account, recent activity, and key metrics.

### Campaigns
Create and manage marketing campaigns, email sequences, and automation workflows.

### Users
Manage user accounts, permissions, and access controls.

### Webhooks
Configure webhook endpoints for real-time integrations with Facebook, email services, and third-party platforms.

### Bot Flows
Design and deploy conversational bot flows for customer engagement and lead qualification.

## Getting Started

1. **Login** to the dashboard at [https://platapayph.innoserver.cloud](https://platapayph.innoserver.cloud)
2. **Navigate** to the desired section using the sidebar
3. **Configure** your settings and integrations
4. **Launch** your campaigns and automations
