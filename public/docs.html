OAuth Account Management

Connect multiple accounts from different providers. Support for personal + work accounts.

Admin API Endpoints

POST /admin/oauth/url

Generate OAuth URL with PKCE for authentication.

Request Body
{
  "provider": "antigravity",
  "accountName": "Personal Gmail"
}
Response
{
  "url": "https://accounts.google.com/o/oauth2/v2/auth...",
  "state": "xyz123...",
  "expiresIn": 300
}
POST /admin/oauth/exchange

Exchange authorization code for access tokens.

Request Body
{
  "code": "4/0A...",
  "state": "xyz123...",
  "callbackUrl": "http://localhost:51121/callback?code=...",
  "accountName": "Personal Gmail"
}
GET /admin/accounts

List all connected OAuth accounts.

Python

# Install
pip install openai

# Usage
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_API_KEY",
    base_url="https://ai-gateway.innoserver.cloud/v1"
)

# Chat
response = client.chat.completions.create(
    model="inno-ai-boyong-4.5",
    messages=[{"role": "user", "content": "Hello!"}]
)

# Streaming
for chunk in client.chat.completions.create(
    model="inno-ai-boyong-4.5",
    messages=[{"role": "user", "content": "Tell me a story"}],
    stream=True
):
    print(chunk.choices[0].delta.content, end="")

# Images
image = client.images.generate(
    model="image-3",
    prompt="A futuristic city",
    size="1024x1024"
)

Node.js

// Install
npm install openai

// Usage
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'YOUR_API_KEY',
  baseURL: 'https://ai-gateway.innoserver.cloud/v1'
});

// Chat
const response = await client.chat.completions.create({
  model: 'inno-ai-boyong-4.5',
  messages: [{ role: 'user', content: 'Hello!' }]
});

console.log(response.choices[0].message.content);
